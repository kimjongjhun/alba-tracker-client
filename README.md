# Alba Tracker Client

This app is meant to keep track of part time work, alba, and provide some stats in an easy to read and understand format.

## Built With

- [React](https://react.dev/)
- [Vite](https://vitejs.dev/)
- [TypeScript](https://vitejs.dev/)
- [Material UI](https://mui.com/)
